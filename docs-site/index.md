## 誰のためのアプリ？

**小説らいたー**は、物語を書きたいすべての人のためのアプリです。

- **初心者の方**: シンプルモードで、余計な機能に迷わず執筆に集中できます
- **プロ志向の方**: 標準モードで、プロットやタイムラインを駆使した緻密な構成管理が可能
- **AIに興味がある方**: 最新のAI技術を活用した執筆体験を提供します

## クイックスタート

1. [アプリにアクセス](https://ai.studio/apps/drive/1LOOn2JjF32z_mmHG2Hy3oblU6AyNwWw_)
2. 「新規プロジェクト作成」をクリック
3. モード（シンプル/標準）を選択してタイトルを入力
4. 執筆開始！

詳しい使い方は[ユーザーガイド](/user-guide/introduction)をご覧ください。
