# キャラクターを作る

このページでは、登場人物（キャラクター）を作成・管理する方法を説明します。

## キャラクター作成の重要性

物語の面白さの半分以上はキャラクターの魅力で決まります。

作者の頭の中にある「なんとなくのイメージ」を、名前、年齢、容姿、そして心の内面まで細かく言葉に落とし込むことで、キャラクターは初めて記号から一人の人間へと変わります。

## キャラクターを新規作成

1. 左パネルの「キャラクター」横にある「＋」ボタンをクリック
2. キャラクター設定モーダルが開く
3. 各項目を入力
4. 「保存」ボタンで保存

## 基本プロフィール

### 必須項目

| 項目 | 説明 |
|------|------|
| 名前 | キャラクターの名前 |
| ふりがな | 読み方（ルビ生成に使用） |

### オプション項目

| 項目 | 説明 |
|------|------|
| 年齢 | キャラクターの年齢 |
| 性別 | 性別 |
| 種族 | 人間、エルフ、AIなど |
| 一人称 | 私、俺、僕など |
| 出身地 | 世界観設定から選択可能 |
| 所属 | 世界観設定から選択可能 |

::: tip 世界観との連携
「出身地」や「所属」を世界観設定で作った場所や組織から選択できます。これにより、キャラクターが物語の世界と密接に繋がっている実感が持てます。
:::

## テーマカラー

キャラクターのテーマカラーを設定できます。

### 用途

- 本文中のセリフの色に反映
- 相関図の枠線の色
- 視覚的にそのキャラを認識するための印

### 自動色付け

AI設定で「セリフに話者の色を適用」をONにすると、AIが書いたセリフに自動的にテーマカラーが適用されます。

## 容姿設定

### 容姿特徴リスト

髪の色や瞳の形など、忘れがちな外見のディテールを箇条書きで管理できます。

例：
- 髪色: 銀色
- 瞳: 琥珀色
- 特徴: 左頬に小さな傷跡

### AIで立ち絵を生成

言葉で入力した特徴を元に、AIがキャラクターのイラストを生成してくれます。

1. 「AIで立ち絵を生成」ボタンをクリック
2. 特徴を入力（容姿設定から自動で入力される）
3. 「生成」ボタンをクリック
4. 生成されたイラストを確認・保存

::: tip 画像のアップロード
自分で用意したイラストをアップロードすることもできます。
:::

## 性格・内面

### 詳細設定

キャラクターの性格、口調、秘密などを設定します。

**「詳細設定」に書いた内容はAIも読み取ります。**

AIが続きを書く際の「性格のガイドライン」になるため、重要な設定はここに書きましょう。

例：
- 性格: 明るく社交的だが、本当に大切なことは誰にも言えない
- 口調: ～だよ、～かな？と柔らかい言い回し
- 口癖: 「まあ、なんとかなるよ」

### メモ

**作者本人だけが見られる秘密のノート**です。

AIにすら知られたくないプロットの構想を書き留めておくことができます。

例：
- 実は最後に裏切る予定
- 3章で死亡させる構想
- 主人公の本当の父親

## AIアシスタントで作成

キャラクター設定モーダルの右下にある「AIアシスタント」ボタンを押すと、AIと対話しながら設定を作成できます。

### 使い方

1. 「AIアシスタント」ボタンをクリック
2. 入力欄に「明るい性格の魔法使いの女の子を作りたい」と入力
3. AIが質問を返してくる
4. 対話を繰り返すと、生成プレビューに設定が埋まっていく
5. 「内容を反映する」ボタンで入力フォームに反映

### 2つのモード

| モード | 説明 |
|--------|------|
| 相談する | アイデア出しのみ、データは変更されない |
| 設定に反映 | 会話内容がプレビューに反映される |

## キャラクター相関図

作成したキャラクターたちの関係を視覚化できます。

### 相関図を開く

左パネルの「相関図を表示」ボタンをクリック

### 関係を追加

1. 「関係を追加」モードにする
2. 関係元のキャラクターをクリック
3. 関係先のキャラクターをクリック
4. 関係性（親友、宿敵、片思いなど）を入力
5. 呼び方（呼び捨て、〇〇先輩など）を設定

### 呼び方の活用

キャラクターが相手をどう呼ぶかを記録しておくと、AIは本文を書く際にもその呼び方を正確に守ってくれます。

例：
- AはBを「お兄ちゃん」と呼ぶ
- BはAを「呼び捨て」で呼ぶ

### ノードの配置

キャラクターのアイコンはドラッグして自由に配置できます。

- 仲の良いグループを近くに置く
- 対立する勢力を左右に分ける

## キャラクター一覧

左パネルのキャラクター欄には、作成したすべてのキャラクターが表示されます。

### 操作

| ボタン | 機能 |
|--------|------|
| 編集 | 設定モーダルを開く |
| ゴミ箱 | キャラクターを削除 |

::: warning 削除の注意
キャラクターを削除すると、そのキャラクターに関連する相関図の関係も削除されます。
:::
