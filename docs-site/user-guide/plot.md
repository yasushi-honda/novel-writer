# プロット・タイムライン

このページでは、プロットとタイムライン機能の使い方を説明します。

## プロット機能

### プロットとは

プロット機能は、物語の「骨組み」や「あらすじ」をカード化して管理する仕組みです。

「ここで誰が出会う」「ここで事件が起きる」といった断片的な出来事を形にするためのものです。

### なぜプロットが必要か

物語の全体像（設計図）が頭の中にしかない状態だと：

- 執筆中に迷子になる
- 話のつじつまが合わなくなる
- 何を書けばいいかわからなくなる

プロットがあれば、何を書くべきか決まっているので、あとは文章で埋めていくだけです。

## プロットを作成

1. 左パネルの「プロット」横にある「＋」ボタンをクリック
2. プロットボードモーダルが開く
3. 各項目を入力
4. 「保存」ボタンで保存

### 入力項目

| 項目 | 説明 |
|------|------|
| タイトル | プロットの見出し |
| 要約・説明 | その場面で起きることや目的 |
| 種別 | 第一章、クライマックス、伏線など |
| 色 | 視覚的な識別用 |

## 種別と色

プロットを「種別」と「色」で分けることで、物語のどの部分がどの役割を担っているのかが視覚的にわかります。

### 種別の例

- 章のまとめ
- クライマックス
- 伏線
- 日常パート
- 転換点

### 色の活用

同じ種別に同じ色を割り当てると、プロットボード上で一目で識別できます。

## プロットボード

### 開き方

左パネルの「すべて表示する（プロット）」ボタンをクリック

### ボードの機能

すべてのプロットカードを広いキャンバスの上に並べ、繋がりを視覚的にデザインできます。

### 関係の追加

カード間に矢印を引いて因果関係を定義します：

1. 「関係を追加」モードにする
2. 原因となるカードをクリック
3. 結果となるカードをクリック
4. 関係にラベルを付ける（例：「引き起こす」「伏線」）

### 関係の削除

不要になった繋がりは「関係の削除」で消せます。

### カードの配置

カードをドラッグして自由に配置できます：

- 原因を左、結果を右に置く
- 時系列で上から下に並べる
- 章ごとにグループ化する

## タイムライン機能

### タイムラインとは

タイムライン機能は、物語の中で「いつ」「どこで」「誰が」「何をしたか」という出来事の流れを、巨大な年表として管理する仕組みです。

### なぜタイムラインが必要か

執筆が進むにつれて、時間の経過に伴う論理的なミスが発生しやすくなります：

- 「昨日の夜に事件が起きたはずなのに、翌朝にはもう犯人が捕まっている」
- 「キャラAが別の場所にいるはずの時間に、ここにいる」

タイムラインで時系列を管理することで、こうした矛盾を防げます。

## タイムラインを開く

左パネルの「タイムラインボードを開く」ボタンをクリック

## レーン

タイムライン上で情報を整理するための列です。

### レーンの例

- 主人公の行動
- 敵組織の動き
- 世界情勢
- サブキャラAの行動

### レーンの効果

同じ時期にそれぞれの場所で何が起きていたのかを並列で把握できます。

## イベントを作成

1. タイムライン上の「＋」ボタンをクリック
2. イベント作成フォームが開く
3. 各項目を入力
4. 保存

### 入力項目

| 項目 | 説明 |
|------|------|
| タイトル | イベントの見出し |
| 時期 | いつ起きたか |
| 発生場所 | どこで起きたか（世界観設定から選択可能） |
| 詳細 | 何が起きたか |
| レーン | どのレーンに配置するか |

## 発生場所の連携

「発生場所」を世界観設定から選択できます。

- 場所をクリックすると設定を確認できる
- 新しい場所を入力すると、世界観設定に追加できる

## イベントの操作

### 移動

イベントカードをドラッグして、時期を入れ替えたり別のレーンに移動したりできます。

### 編集

カードをダブルクリックして編集

### 削除

カードの削除ボタンをクリック

## プロットとタイムラインの連携

### プロットからタイムラインへ

プロットボードで「タイムラインへ送る」ボタンをクリックすると、プロットの内容をタイムラインに反映できます。

これにより：
- 構成案（やりたいこと）→ 実際の時間軸（起きたこと）
という変換がスムーズに行えます。

### タイムラインからプロットへ

タイムラインのイベントをプロットに紐付けることもできます。

## AIでプロットをまとめる

自分が書いた混沌とした文章を、AIが理路整然としたプロットカードに整理してくれます。

### 使い方

1. 右パネルで「相談モード」を選択
2. 「ここまでの内容をプロットとしてまとめて」と入力
3. AIが提案したプロットを確認
4. 「承認」ボタンでプロットボードに追加

## アウトライン

本文の章構成を管理する機能です。

### アウトラインの開き方

左パネルのタブを「アウトライン」に切り替え

### 機能

| 機能 | 説明 |
|------|------|
| 章へジャンプ | タイトルをクリックで該当箇所に移動 |
| 順番入れ替え | ドラッグ&ドロップで章を移動 |
| 章の設定 | タイトルやメモを編集 |
| 章の追加 | 新しい章を末尾に追加 |
| 章の削除 | 章全体を削除 |

### 見出しとの連携

本文中で見出し（`# 第1章` など）を作ると、自動的にアウトラインに登録されます。

## ベストプラクティス

### 1. 大まかなプロットから

最初は大まかな流れ（起承転結）だけ作り、書き進めながら詳細を追加

### 2. タイムラインは必要に応じて

シンプルな物語では不要。複数の視点や複雑な時系列がある場合に活用

### 3. プロットとタイムラインの使い分け

- **プロット**: 物語の構成・流れ
- **タイムライン**: 時間の整合性
