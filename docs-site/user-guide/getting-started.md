# 最初の一歩

このページでは、小説らいたーを使い始めるための基本的な手順を説明します。

## アプリにアクセス

小説らいたーはWebアプリケーションです。ブラウザからアクセスして使用します。

::: info 対応ブラウザ
- Google Chrome（推奨）
- Microsoft Edge
- Firefox
- Safari
:::

## プロジェクトを作成する

### 1. 新規プロジェクト作成

アプリを開くと、プロジェクト選択画面が表示されます。

1. **モードを選択**
   - **シンプルモード**: 初心者向け。必要最低限の機能だけが表示されます
   - **標準モード**: すべての機能が使えます

2. **タイトルを入力**
   - 物語のタイトルを入力します（後から変更可能）

3. **「作成」ボタンをクリック**
   - 新しいプロジェクトが作成され、執筆画面に移動します

### 2. 既存プロジェクトを開く

過去に作成したプロジェクトは、プロジェクト選択画面のリストに表示されます。「開く」ボタンをクリックして開きます。

### 3. プロジェクトをインポート

他の環境で作成したプロジェクトファイル（`.json`）を読み込むことができます。

## シンプルモードと標準モード

2つのモードが存在する背景には、「書くことに集中したいとき」と「物語の矛盾を整理したいとき」では、人間の脳が使うモードが異なるという思想があります。

### シンプルモード

- 画面からはナレッジやプロット、タイムラインといった複雑なツールが消えます
- 残るのは原稿用紙（本文エディタ）と、最低限のキャラクター・世界観のリストだけ
- 「今は細かいことはいいから、とにかく物語の続きを書きたい！」という瞬間に最適

::: tip こんな人におすすめ
- スマホで電車の中で書くとき
- 深夜の勢いで書き進めたいとき
- 初めてアプリを使う人
:::

### 標準モード

- このアプリのフルパワーを発揮できる状態
- 左側にキャラクターや世界観、プロット、タイムラインといった「資料棚」が並びます
- 設定を読み返しながら緻密に物語を組み上げたい人向け

::: tip こんな人におすすめ
- 複雑な設定を持つ物語を書く人
- AIに自分の作った設定を完璧に守らせたい人
- 長編小説に挑戦したい人
:::

## モードの切り替え

ヘッダー部分にある「シンプルモードへ」「標準モードへ」ボタンで、いつでもモードを切り替えられます。

データはすべて保持されるので、安心して切り替えてください。

## プロジェクト設定

### プロフィール設定

- **アイコン**: あなたの分身となるアイコンを設定
- **名前**: AIに呼んでほしい名前を設定

AIがあなたを「先生」や「相棒」と呼ぶようになることで、孤独になりがちな執筆作業の中に「誰かが自分を見てくれている」という安心感が生まれます。

### ユーザーモード

| モード | 説明 |
|--------|------|
| かんたん | 不要なボタンを隠して画面をスッキリ |
| 標準 | すべての機能がバランスよく配置 |
| プロ | 操作履歴の管理など、熟練者向けの機能が使用可能 |

### 文章スタイル設定

- **視点**: 一人称 / 三人称
- **トーン**: 文体の雰囲気
- **創作性**: AIにどれくらい大胆なアイデアを出させるか
