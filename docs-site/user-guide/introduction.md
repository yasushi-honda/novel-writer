# イントロダクション

「小説らいたー」へようこそ！

このドキュメントは、物語を紡ぐすべての人へ向けて、「小説らいたー」という道具がどのようにあなたの創作を支え、どのような仕組みで動いているのかを、一から十まで解き明かすものです。

## 小説らいたーとは

**小説らいたー**は、AI搭載の小説執筆支援アプリケーションです。

単に文章を入力するためのソフトではありません。これは、あなたの頭の中にだけ存在する「美しく、時に残酷で、愛おしい物語の世界」を、この現実世界へと安全に引き出すための「ゆりかご」のような道具です。

## なぜこのアプリが生まれたのか

物語を書くということは、時にとても孤独で、自分の才能を疑い、真っ白な画面に絶望してしまうような、苦しい作業でもあります。

このアプリは、そんな「創作の孤独と停滞」という問題を解決するために生まれました。

AIはあなたの代わりにお話を完成させる存在ではありません。あなたが迷ったときに灯をともし、疲れたときに背中を押し、自信をなくしたときに一番のファンとして拍手を送る、そんな「伴走者」です。

## 3つの約束

### 1. 段階的に学べる

一度に全部を理解する必要はありません。物語が育っていくのと同じように、あなたとこのアプリの関係もゆっくりと育っていきます。

最初はただの日記帳のように使い始めてもいい。少し慣れてきたらキャラクターを登録し、物語が複雑になってきたら相関図やタイムラインの力を借りればいい。

### 2. 失敗を恐れない

元に戻す（Undo）機能が充実しているので、何度でもやり直せます。とりあえず書いてみる、とりあえずAIに任せてみる。ダメだったら戻せばいい。

### 3. あなたが主役

AIはあくまで「アシスタント」です。物語の「核」になる部分や、キャラクターの決定的な一言は、やはり作者であるあなた自身の手で生み出されるべきです。

## 対象読者

このドキュメントは、以下のような方を想定しています：

- **初めて小説を書く方**: 基本的な使い方から丁寧に説明します
- **他のツールから移行する方**: インポート機能で既存の原稿を取り込めます
- **AI執筆ツールに興味がある方**: 最新のAI技術を活用した執筆体験を提供します

::: tip ヒント
迷ったら、まずは「シンプルモード」で始めましょう。必要な機能だけが表示されるので、操作に迷うことがありません。
:::
