# ナレッジベース

このページでは、ナレッジベース機能の使い方を説明します。

## ナレッジとは

ナレッジ機能は、物語の中に登場する独自の用語、アイテム、歴史、魔法の呪文などをまとめた「作中事典（Wiki）」です。

キャラクター設定や世界観設定よりもさらに細かく、断片的な知識を保存するために存在します。

## なぜナレッジが必要か

長編小説を書いていると、細かいディテールを忘れてしまいがちです：

- 「あの薬草の名前なんだっけ？」
- 「あの魔法を使うのに必要な条件は？」
- 「あのアイテムの効果は？」

ナレッジ機能は、設定の「ど忘れ」による作業中断をゼロにします。

## ナレッジを作成

1. 左パネルの「ナレッジベース」横にある「＋」ボタンをクリック
2. ナレッジ設定モーダルが開く
3. 各項目を入力
4. 「保存」ボタンで保存

## 入力項目

| 項目 | 説明 |
|------|------|
| 名前 | ナレッジの名前（例：太陽の剣） |
| 内容 | 詳細な説明 |
| カテゴリ | 分類（アイテム、魔法、歴史など） |
| タグ | 検索用のタグ（複数設定可能） |

## 内容の重要性

**「内容」欄に書いたことは、AIにとっての「絶対的な教科書」になります。**

例えば「太陽の剣」というナレッジを作り、その内容に「持ち主以外が触れると灰になる」と書いておけば、AIは勝手に敵キャラクターにその剣を握らせるような展開を書かなくなります。

## カテゴリとタグ

### カテゴリ

大まかな分類に使います：

- アイテム
- 魔法
- 歴史
- 人物
- 場所
- 用語

### タグ

より細かい分類や横断的な整理に使います：

- 重要
- 未回収の伏線
- 第1章関連
- メインキャラ関連

## ナレッジのリンク

本文中でナレッジ名を `[[ナレッジ名]]` のように二重のカッコで囲むと、クリック可能なリンクになります。

```
勇者は[[太陽の剣]]を手に取った。
```

### リンクの効果

- クリックでナレッジの内容を確認
- その場で編集も可能
- 執筆画面を離れることなく設定を確認できる

## ピン留め

重要なナレッジをピン留めすると、AIが特に優先して守るべき情報として扱います。

### ピン留め方法

ナレッジカードの画鋲マークをクリック

### ピン留めの効果

- リストの最上部に固定表示
- AIが執筆時に「特に優先して守るべき絶対のルール」として扱う

## ナレッジ一覧

### 開き方

左パネルの「すべて表示して検索」ボタンをクリック

### 一覧画面の機能

すべてのナレッジがカード形式で表示されます。

| 機能 | 説明 |
|------|------|
| 検索 | 名前だけでなく説明文の中身も検索 |
| タグ検索 | 特定のタグを持つ項目だけ表示 |
| カテゴリフィルタ | 特定のカテゴリだけ表示 |
| ピン管理 | ピン留めの切り替え |

## AIからの提案

AIが執筆中に新しい用語を見つけると、ナレッジとして提案してくれることがあります。

### 提案の承認/却下

右パネルに表示される提案を確認し：

- **承認**: ナレッジベースに追加
- **却下**: 提案を破棄

## ベストプラクティス

### 1. 書きながら追加

最初からすべてを登録する必要はありません。執筆中に「これは後で使いそうだな」と思ったら追加しましょう。

### 2. 一貫性のために活用

同じアイテムや魔法を複数回書くとき、ナレッジを参照することで表記ゆれを防げます。

### 3. 伏線管理に活用

「未回収」タグを付けた伏線を一覧で確認し、回収漏れを防ぎましょう。

### 4. AIへの指示として活用

「この魔法は〜という制限がある」という設定をナレッジに書いておくと、AIがその制限を守って執筆してくれます。

## 固有名詞ジェネレーター

名前が思いつかないときは、固有名詞ジェネレーターが便利です。

### 使い方

1. 左パネルの「ジェネレーターを開く」ボタンをクリック
2. カテゴリを選択（ファンタジー風、現代日本風、地名、技名など）
3. キーワードを入力（例：「炎」「気高い」）
4. 「生成」ボタンをクリック
5. 10個の候補から選択

### 生成された名前の活用

- コピーボタンでクリップボードにコピー
- そのままナレッジやキャラクター設定に反映
