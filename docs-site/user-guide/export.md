# 書き出し

このページでは、作成した物語を外部に書き出す方法を説明します。

## 書き出し形式

小説らいたーでは、3つの形式で書き出しができます。

| 形式 | 用途 | 特徴 |
|------|------|------|
| JSON | バックアップ | すべてのデータを含む |
| TXT | 汎用テキスト | 装飾なし、どこでも開ける |
| HTML | 装飾付き出力 | 色やルビを保持 |

## JSON形式（バックアップ）

### 用途

- プロジェクトの完全バックアップ
- 別の環境への移行
- データの安全な保管

### 含まれるデータ

- 本文
- キャラクター設定
- 世界観設定
- ナレッジベース
- プロット・タイムライン
- 相関図の配置
- AI設定
- チャット履歴

### 書き出し方法

1. ヘッダーの「エクスポート」ボタンをクリック
2. `プロジェクト名.json` ファイルがダウンロードされる

### 読み込み方法

1. プロジェクト選択画面を開く
2. 「プロジェクトをインポート」ボタンをクリック
3. JSONファイルを選択
4. プロジェクトが復元される

::: warning バージョン互換性
古いバージョンで書き出したJSONファイルは、新しいバージョンで正常に読み込めない場合があります。
:::

## TXT形式（プレーンテキスト）

### 用途

- 公募への投稿
- 他のエディタでの編集
- シンプルなテキストとしての利用

### 特徴

- 本文のみが出力される
- 装飾（太字、色、ルビ）は消える
- どんな環境でも開ける

### 書き出し方法

1. ヘッダーの「.txtで出力」ボタンをクリック
2. `プロジェクト名.txt` ファイルがダウンロードされる

### 注意点

- キャラクター設定などは含まれない
- 装飾情報は失われる
- 章の区切り（見出し）は保持される

## HTML形式（装飾付き）

### 用途

- 装飾を保持した出力
- 電子書籍風の見た目
- Webでの公開

### 特徴

- 文字色・ルビが保持される
- 章立ての構造が維持される
- 目次が自動生成される
- キャラクター紹介を含められる

### 書き出し方法

1. ヘッダーの「.htmlで出力」ボタンをクリック
2. 書き出し設定モーダルが開く
3. オプションを選択
4. 「書き出す」ボタンをクリック
5. `プロジェクト名.html` ファイルがダウンロードされる

### 書き出しオプション

| オプション | 説明 |
|-----------|------|
| 目次を含める | 章ごとの目次を先頭に追加 |
| キャラクター紹介 | 登場人物の紹介ページを追加 |
| 立ち絵を含める | キャラクターの画像を含める |
| テーマ | ライト/ダーク/セピアの外観 |

### HTML出力の活用

- ブラウザで開いて読む
- 印刷してPDF化する
- Webサイトに掲載する

## プレビュー機能

書き出す前に、本文がどのように見えるか確認できます。

### 使い方

1. ヘッダーの「プレビュー」ボタンをクリック
2. プレビューモーダルが開く
3. 装飾が適用された状態で確認
4. 閉じるで戻る

### プレビューでできること

- 装飾の確認（色、ルビなど）
- 全体の見通し確認
- 章構成の確認

## 定期的なバックアップ

### なぜバックアップが必要か

- ブラウザのデータはいつか消える可能性がある
- パソコンの故障に備える
- 作業の区切りで状態を保存

### おすすめのバックアップ頻度

| 状況 | 頻度 |
|------|------|
| 執筆中 | 毎日の終わりに |
| 重要な変更後 | その都度 |
| 長期保管 | 週1回程度 |

### バックアップの保管場所

- クラウドストレージ（Google Drive、Dropboxなど）
- 外付けHDD/SSD
- 複数の場所に分散保管がベスト

## 自動保存

小説らいたーは、操作するたびに自動でブラウザに保存されます。

### 確認方法

画面右上に保存状態が表示されます：

- **保存済み**: すべて保存されている
- **保存中...**: 保存処理中
- **未保存**: 保存されていない（まれ）

### 注意点

- 自動保存はブラウザ内のみ
- ブラウザのデータを消すと消える
- 定期的なJSONバックアップを推奨

## バージョン管理（スナップショット）

プロジェクトの状態を任意のタイミングで保存・復元できます。

### スナップショットを保存

1. 左パネルの「バージョン管理」タブを開く
2. 「スナップショットを保存」ボタンをクリック
3. 名前とメモを入力
4. 「保存」をクリック

### スナップショットを復元

1. 「スナップショットを管理」ボタンをクリック
2. 復元したいスナップショットを選択
3. 「復元」をクリック

### 活用シーン

- 大きな変更前に保存
- 分岐点で両方の展開を試す
- 過去のバージョンに戻りたいとき
