# 本文を書く

このページでは、物語の本文を執筆する方法を説明します。

## 直接入力

白い画面にキーボードで文字を打ち込む「直接入力」は、あなたの魂を物語に吹き込む最も神聖な行為です。

### 入力方法

1. 画面下部の「本文を直接入力」エリアをクリック
2. テキストエリアが展開される
3. 文章を入力
4. 「本文に追加」ボタンをクリック

入力した文章は、物語の最新の1ページとして積み上げられます。

### AIとの連携

あなたが書いた最新の文章をAIは即座に読み込みます。

例えば、あなたが「リアは静かに涙を流した」と書けば、AIはその「悲しみ」の余韻を汲み取って、次の場面を提案してくれます。

## テキスト装飾

本文を編集する際、文字を装飾することで「思考補助」になります。

### 太字・下線

そのシーンで最も重要なキーワードや、後の伏線になるアイテムを強調します。

```markdown
**重要なキーワード**
__下線付きのテキスト__
```

### 見出し

物語を章ごとに区切り、長い旅路に道標を立てます。

```markdown
# 第1章 旅立ちの朝
```

見出しを作成すると、自動的にアウトライン（目次）に登録されます。

### ルビ（ふりがな）

ファンタジーの固有名詞に特別な読みを与えたり、詩的な表現を可能にします。

```markdown
《強敵|とも》と呼べる存在
```

### 文字色変更

会話文において威力を発揮します。キャラクターのテーマカラーに合わせてセリフの色を変えることで、誰が話しているかが一目でわかります。

::: tip 自動色付け
キャラクター設定でテーマカラーを設定し、AI設定で「セリフに話者の色を適用」をONにすると、AIが書いたセリフに自動的に色が付きます。
:::

## 段落の編集

既存の段落を編集するには：

1. 編集したい段落にマウスを合わせる
2. 表示される「編集」ボタンをクリック
3. テキストを編集
4. 「保存」ボタンで確定、「キャンセル」で破棄

### 編集ツールバー

編集モードでは、以下のツールが使えます：

| ツール | 機能 | 操作 |
|--------|------|------|
| 太字 | テキストを強調 | テキスト選択後クリック |
| 下線 | 下線を追加 | テキスト選択後クリック |
| 見出し | 章タイトルに変換 | 行頭で使用 |
| ルビ | ふりがなを追加 | テキスト選択後クリック |
| 文字色 | 色を変更 | テキスト選択後クリック |

## 段落のピン留め

重要な段落をピン留めすると、AIがその内容を特に重視して執筆します。

- マウスオーバーで表示される「ピン留め」ボタンをクリック
- ピン留めされた段落は特別なマークで表示される

::: warning 注意
ピン留めされた情報は、AIにとって「絶対的な教科書」になります。誤った情報をピン留めすると、AIの出力にも影響します。
:::

## AIと一緒に書く

「自分で書く」ことと「AIに任せる」ことのバランスが自由であることが、このアプリの気持ちよさの源泉です。

### 相談モード

- AIは本文を一行も書かない
- アイデア出しや壁打ちに最適
- 「次の展開を3つ出して」「このキャラの過去を一緒に考えて」

### 執筆モード

- AIが直接本文を書く
- 「ここから二人が仲直りするシーンを書いて」
- 物語の続きがスラスラと生み出される

### バランスのとり方

```
あなたの書いた文章
    ↓
AIが続きを提案
    ↓
気に入った部分を採用、必要なら手直し
    ↓
あなたがさらに書き足す
    ↓
（繰り返し）
```

## 複数提案機能

「複数提案」をONにすると、AIは一度に3パターンの異なる展開を提案します。

### 使い方

1. 右パネルの「複数提案」トグルをON
2. AIに指示を送信
3. 複数の展開案が提示される
4. 気に入ったものを「この展開を採用」で選択

### メリット

- 「唯一の正解」を探す苦しさから解放
- 予想外の面白い展開に出会える
- 却下した案も履歴に残り、後から復活可能

## AI支援メニュー

右パネルの入力欄横にある「＋」ボタンから、よく使う指示をワンクリックで入力できます。

### 執筆支援

- 続きを書いて
- 情景描写を足して
- キャラの心理をもっと深く

### 推敲・校正

- 誤字脱字をチェック
- 語尾の重複を直して
- 文章を読みやすく

### アイデア相談

- 次の展開の案を3つ出して
- 設定の矛盾がないか確認して

### 特殊生成

- 章タイトルの案を出して
- SNS投稿用の140字あらすじ

## 元に戻す／やり直す

操作を間違えても大丈夫。いつでも元に戻せます。

| 操作 | ショートカット |
|------|--------------|
| 元に戻す | `Ctrl+Z` |
| やり直す | `Ctrl+Y` |

### 対象範囲の設定

プロジェクト設定で、Undoの対象範囲を選べます：

- **すべての操作**: 本文、設定、チャットすべて
- **本文だけ**: 本文の変更のみ
- **AIチャットだけ**: チャット履歴のみ
