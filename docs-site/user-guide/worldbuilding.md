# 世界観を作る

このページでは、物語の舞台となる世界観を構築する方法を説明します。

## 世界観構築の重要性

一貫性のある世界観こそが、読者を物語に没入させるための魔法の鍵です。

適当に決めた設定は、物語が進むにつれて必ず矛盾を生みます。この機能は、設定の作り込みが甘くて途中で筆が止まってしまう人や、ファンタジーやSFのような複雑な設定を必要とする作家のために用意されました。

## 世界観設定を新規作成

1. 左パネルの「世界観」横にある「＋」ボタンをクリック
2. 世界観設定モーダルが開く
3. テンプレートを選択、または自由に項目を追加
4. 各項目を入力
5. 「保存」ボタンで保存

## テンプレート

テンプレートを選ぶだけで、その項目を説明するために必要な枠組みが自動的にセットされます。

### 利用可能なテンプレート

| テンプレート | 自動生成される項目 |
|-------------|-----------------|
| 場所 | 所属地域、気候、特産品など |
| 組織 | リーダー、目的、規模など |
| 魔法 | 原理、制限、習得条件など |
| 歴史的事件 | 年代、影響、関係者など |
| アイテム | 効果、入手方法、所有者など |
| 種族 | 特徴、文化、生息地など |

::: tip 自由入力も可能
テンプレートを使わず、自分で好きな項目を追加することもできます。
:::

## 基本情報

### 必須項目

| 項目 | 説明 |
|------|------|
| 名前 | 世界観項目の名前 |
| 種別 | テンプレートまたは自由入力 |

### 詳細設定

世界観の詳細を記述します。

**「詳細設定」に書いた内容はAIも読み取ります。**

例えば「この国では魔法が禁止されている」と書いておくと、AIはそのルールを前提とした文章を書いてくれます。

### メモ

**作者本人だけが見られる秘密のノート**です。

例：
- 実はこの国は3章で滅びる運命にある
- この組織の真の目的は世界征服

## ビジュアル設定

### 地図・イメージ画像

地図やイメージ画像をアップロードし、いつでも拡大して確認できます。

- 位置関係を間違えるミスを防げる
- 物語の世界をより具体的にイメージできる

### アップロード方法

1. 「画像をアップロード」エリアをクリック
2. ファイルを選択
3. 画像が設定される

## カスタム項目

テンプレートにない情報を追加できます。

### 追加方法

1. 「項目を追加」ボタンをクリック
2. 項目名を入力（例：「人口」「通貨」）
3. 値を入力（例：「約100万人」「金貨」）

### 削除方法

各項目の横にある「×」ボタンで削除できます。

## AIアシスタントで作成

世界観設定モーダルの「AIアシスタント」ボタンから、AIと対話しながら設定を作成できます。

### 使い方

1. 「AIアシスタント」ボタンをクリック
2. 「魔法学校のある街を作りたい」と入力
3. AIが質問を返してくる（名前は？気候は？特徴は？）
4. 対話を繰り返すと、設定が埋まっていく
5. 「内容を反映する」ボタンで入力フォームに反映

## 世界観とキャラクターの連携

作成した世界観設定は、キャラクター作成時に以下の項目で選択できます：

- **出身地**: 世界観の「場所」から選択
- **所属**: 世界観の「組織」から選択

これにより、キャラクターと世界観が有機的に繋がります。

## 世界観とタイムラインの連携

タイムラインの「発生場所」で、世界観設定から場所を選択できます。

逆に、タイムラインで新しい場所を入力すると、その場をそのまま世界観設定に追加することもできます。

## 世界観一覧

左パネルの世界観欄には、作成したすべての世界観設定が表示されます。

### 操作

| ボタン | 機能 |
|--------|------|
| 編集 | 設定モーダルを開く |
| ゴミ箱 | 世界観設定を削除 |

### フィルタリング

種別（場所、組織、魔法など）でフィルタリングして表示できます。

## ベストプラクティス

### 1. 大から小へ

まず大きな枠組み（世界全体、大陸、国）を作り、そこから詳細（街、村、建物）へと落とし込んでいくと整理しやすいです。

### 2. 必要に応じて追加

最初からすべてを決める必要はありません。物語を書き進めながら、必要に応じて設定を追加していきましょう。

### 3. 矛盾チェック

AIに「設定の矛盾がないか確認して」と頼むと、整合性をチェックしてくれます。
