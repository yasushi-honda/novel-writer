# AIアシスタント活用法

このページでは、AIアシスタントを最大限に活用する方法を説明します。

## AIアシスタントの基本

AIアシスタントは、あなたの執筆を支援する「伴走者」です。

- 本文を書いてもらう
- アイデアを相談する
- 設定の整合性をチェックしてもらう
- 名前やアイデアを生成してもらう

## 2つのモード

### 相談モード

AIを「優秀な編集者」や「気の合う友人」にするモードです。

**特徴**:
- AIは本文を一行も書かない
- アイデア出しや壁打ちに最適
- 本文が増えないので安心して何度でも話せる

**使いどころ**:
- 次の展開を考える
- キャラクターの過去を一緒に考える
- 設定の矛盾がないかチェック

### 執筆モード

AIを「ゴーストライター」や「共同執筆者」にするモードです。

**特徴**:
- 指示を送ると、AIが小説の文章を生成
- 生成された文章は本文に直接追加される

**使いどころ**:
- 続きを書いてほしいとき
- シーンの詳細を膨らませたいとき
- 描写を追加したいとき

## AI設定

プロジェクト設定からAIの挙動を細かく設定できます。

### 基本設定

| 設定 | 説明 |
|------|------|
| 視点 | 一人称 / 三人称 |
| トーン | 文体の雰囲気 |
| 創作性 | 控えめ / 標準 / 大胆 |
| 長さ | 生成する文章の長さ |

### 高度な設定

| 設定 | 説明 |
|------|------|
| 知識への忠実度 | 設定をどれくらい厳密に守るか |
| 提案頻度 | ナレッジ・プロットの提案頻度 |
| 記憶範囲 | 直近のみ / 要約含む / 全文脈 |

### ペルソナ設定

AIの「性格」を選べます：

- 丁寧
- 親しい
- 厳しい
- など

AIがあなたを呼ぶ名前も設定できます。

### 出力フォーマット

| 設定 | 説明 |
|------|------|
| セリフに話者名を出す | 「アラン「行くぞ！」」形式 |
| セリフに話者の色を適用 | キャラのテーマカラーを反映 |
| 文体模倣 | あなたの文体に寄せる |

## 複数提案機能

「複数提案」をONにすると、AIが一度に複数の展開案を提示します。

### 使い方

1. 右パネルの「複数提案」トグルをON
2. 指示を送信
3. 2〜3パターンの展開案が表示される
4. 気に入ったものを「この展開を採用」

### メリット

- 1つの正解を探すストレスから解放
- 予想外の展開に出会える
- 却下した案も履歴に残る

## 履歴機能

一度却下した提案も、履歴から復活させられます。

### 使い方

1. 右パネルの履歴ボタンをクリック
2. 過去の提案が一覧表示
3. 復活させたい提案を選択

## AI支援メニュー

よく使う指示をワンクリックで入力できます。

### 執筆支援

- **続きを書いて**: 自然に物語を続ける
- **情景描写を足して**: 風景や雰囲気の描写を追加
- **キャラの心理をもっと深く**: 内面描写を充実させる

### 推敲・校正

- **誤字脱字をチェック**: 基本的な間違いを発見
- **語尾の重複を直して**: 読みやすさを向上
- **文章を読みやすく**: 全体的な推敲

### アイデア相談

- **次の展開の案を3つ出して**: 複数の選択肢を提示
- **設定の矛盾がないか確認して**: 整合性チェック
- **このキャラならどう行動する？**: キャラクター性の確認

### 特殊生成

- **章タイトルの案を出して**: タイトル候補を生成
- **SNS用あらすじ**: 140字程度の要約を生成
- **キャッチコピーを考えて**: 宣伝用のフレーズ

## 効果的なプロンプトの書き方

### 具体的に指示する

```
✗ 続きを書いて
◯ 主人公がドアを開けるシーンの続きを、緊張感を持たせて書いて
```

### コンテキストを与える

```
✗ 会話を書いて
◯ 長年の親友同士が久しぶりに再会する場面の会話を書いて。
  お互いに照れくさそうに話す感じで
```

### 制約を明示する

```
✗ 説明を追加して
◯ 魔法の発動シーンを、視覚的な描写を中心に200字程度で追加して
```

## ピン留めの活用

重要な設定をピン留めすると、AIが特に重視します。

### ピン留めすべき情報

- 物語の核心となるルール
- キャラクターの絶対に守るべき設定
- 世界観の重要な制約

### ピン留めの効果

ピン留めされた情報は「絶対的な教科書」として扱われます。

## テキスト解析機能

既存の文章をAIに解析させて、キャラクターや用語を自動抽出できます。

### 使い方

1. 解析したい文章をコピー
2. 解析エリアに貼り付け
3. 「解析」ボタンをクリック
4. AIが人物やキーワードを抽出
5. 反映したい項目を選択して保存

### 活用シーン

- 他のアプリで書いた下書きを取り込む
- 古いノートのメモを整理する
- 思いつくままに書いた文章を構造化する

## よくある質問

### Q: AIが設定を無視した文章を書く

A: 重要な設定はピン留めしてみてください。また、詳細設定に明確に書いておくことも効果的です。

### Q: 生成された文章が長すぎる/短すぎる

A: AI設定の「長さ」を調整してください。

### Q: 文体が自分のものと違う

A: 「文体模倣」をONにすると、あなたの書いた文章の特徴を学習して似せてくれます。

### Q: アイデアがマンネリ化する

A: 「創作性」を「大胆」に設定すると、予想外のアイデアが出やすくなります。
