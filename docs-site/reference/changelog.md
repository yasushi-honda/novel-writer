# 更新履歴

小説らいたーの更新履歴です。

## 最新バージョン

### v1.0.0 (2026-01-XX)

初回リリース

#### 主な機能

- **本文執筆**
  - リッチテキストエディタ（太字、下線、ルビ、文字色）
  - 章・段落の管理
  - 直接入力エリア

- **AI執筆支援**
  - 相談モード / 執筆モード
  - 複数提案機能
  - AI支援メニュー
  - 文体模倣機能

- **キャラクター管理**
  - キャラクター設定（プロフィール、容姿、性格）
  - AIによる立ち絵生成
  - キャラクター相関図
  - テーマカラー設定

- **世界観構築**
  - テンプレート機能（場所、組織、魔法など）
  - カスタム項目
  - 地図・イメージ画像

- **ナレッジベース**
  - 作中辞書機能
  - タグ・カテゴリ管理
  - ピン留め機能
  - 本文内リンク

- **プロット・タイムライン**
  - プロットカード管理
  - 関係線による因果関係表示
  - タイムラインによる時系列管理
  - レーンによる並列管理

- **書き出し**
  - JSON（バックアップ）
  - TXT（プレーンテキスト）
  - HTML（装飾付き）

- **その他**
  - シンプルモード / 標準モード
  - 表示設定（テーマ、フォント、文字サイズ）
  - Undo/Redo（スコープ選択可能）
  - 全体検索
  - コマンドパレット

## フィードバック

バグ報告や機能リクエストは、GitHubのIssueページでお願いします。

[GitHub Issues](https://github.com/yasushi-honda/novel-writer/issues)

### バグ報告時のお願い

バグを報告する際は、以下の情報を含めていただけると助かります：

1. **発生した問題の説明**
2. **再現手順**
3. **期待される動作**
4. **実際の動作**
5. **環境情報**（ブラウザ、OS）
6. **スクリーンショット**（あれば）

### 機能リクエスト

新機能のリクエストも歓迎します。以下の情報を含めていただけると検討しやすいです：

1. **どんな機能が欲しいか**
2. **なぜその機能が必要か**
3. **どのように使いたいか**
