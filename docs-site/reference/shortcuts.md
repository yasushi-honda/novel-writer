# キーボードショートカット

小説らいたーで使用できるキーボードショートカットの一覧です。

## グローバルショートカット

どの画面でも使用できるショートカットです。

| ショートカット | 機能 |
|--------------|------|
| `Ctrl + Z` | 元に戻す（Undo） |
| `Ctrl + Y` | やり直す（Redo） |
| `Ctrl + Shift + Z` | やり直す（Redo）※代替 |
| `Ctrl + S` | 保存（自動保存のため通常不要） |
| `Ctrl + F` | 全体検索を開く |
| `Ctrl + K` | コマンドパレットを開く |
| `Escape` | モーダルを閉じる / 操作をキャンセル |

## サイドバー操作

| ショートカット | 機能 |
|--------------|------|
| `Ctrl + [` | 左サイドバーの開閉 |
| `Ctrl + ]` | 右サイドバーの開閉 |

## テキスト編集

本文の編集モード中に使用できるショートカットです。

| ショートカット | 機能 |
|--------------|------|
| `Ctrl + B` | 太字 |
| `Ctrl + U` | 下線 |
| `Ctrl + Enter` | 編集を確定して保存 |

## AIアシスタント

| ショートカット | 機能 |
|--------------|------|
| `Ctrl + Enter` | メッセージを送信 |

## モーダル内

| ショートカット | 機能 |
|--------------|------|
| `Escape` | モーダルを閉じる（未保存の変更がある場合は確認） |
| `Ctrl + Enter` | 保存して閉じる |

## コマンドパレット

`Ctrl + K` でコマンドパレットを開くと、以下の操作をキーボードだけで実行できます。

### 利用可能なコマンド

| コマンド | 説明 |
|---------|------|
| 相関図を開く | キャラクター相関図を表示 |
| タイムラインを開く | タイムラインボードを表示 |
| プロットボードを開く | プロットボードを表示 |
| ナレッジ一覧を開く | ナレッジベース全体を表示 |
| プレビュー | 本文のプレビューを表示 |
| 設定を開く | プロジェクト設定を表示 |
| シンプルモードへ | モードを切り替え |
| 標準モードへ | モードを切り替え |

### 使い方

1. `Ctrl + K` でパレットを開く
2. コマンド名の一部を入力
3. 矢印キーで選択
4. `Enter` で実行

## 全体検索

`Ctrl + F` で全体検索を開くと、プロジェクト内のすべてのテキストを検索できます。

### 検索対象

- 本文
- キャラクター名・設定
- 世界観名・設定
- ナレッジ名・内容
- プロット

### 検索操作

| 操作 | 説明 |
|------|------|
| 入力 | 検索キーワードを入力 |
| `Enter` | 次の結果にジャンプ |
| `Shift + Enter` | 前の結果にジャンプ |
| `Escape` | 検索を閉じる |

## ドラッグ操作

マウス操作ですが、重要な操作として記載します。

| 操作 | 場所 | 説明 |
|------|------|------|
| ドラッグ | アウトライン | 章の順番を入れ替え |
| ドラッグ | 相関図 | キャラクターの位置を移動 |
| ドラッグ | タイムライン | イベントの位置を移動 |
| ドラッグ | プロットボード | プロットカードの位置を移動 |
| ドラッグ | サイドバー境界 | サイドバーの幅を調整 |

## Tips

### 効率的な執筆のために

1. **`Ctrl + Enter`** を覚えると、入力 → 送信がスムーズに
2. **`Ctrl + [` / `Ctrl + ]`** でサイドバーを素早く開閉
3. **`Ctrl + K`** のコマンドパレットで機能に素早くアクセス

### よく使う操作の組み合わせ

```
AIに続きを書いてもらう流れ：
1. 指示を入力
2. Ctrl + Enter で送信
3. 結果を確認
4. 必要に応じて Ctrl + Z で元に戻す
```

```
キャラクター設定を確認しながら執筆：
1. Ctrl + [ で左サイドバーを開く
2. キャラクターを確認
3. Ctrl + [ で閉じて執筆に集中
```
